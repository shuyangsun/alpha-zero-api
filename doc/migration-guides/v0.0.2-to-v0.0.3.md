# Migration Guide - v0.0.2 to v0.0.3

Changed project root namespace from `alphazero` to `az`. You may use the
following script to identify places that need to be changed.

```bash
#!/usr/bin/env bash
# Finds all occurrences of "alphazero" (lowercase) in a given directory.
# Usage: ./find-alphazero.sh <directory>

if [[ $# -ne 1 ]]; then
  echo "Usage: $0 <directory>" >&2
  exit 1
fi

if [[ ! -d "$1" ]]; then
  echo "Error: '$1' is not a directory." >&2
  exit 1
fi

grep -rn 'alphazero' "$1"
```

To replace all occurrences, use the following script. By default it performs a
dry run, printing what would change. Pass `--apply` to modify files in place and
run `clang-format` on each changed file.

```bash
#!/usr/bin/env bash
# Replaces "alphazero" with "az" in all files under a given directory.
# Dry-run by default; pass --apply to modify files in place.
# Usage: ./migrate-alphazero.sh <directory> [--apply]

set -euo pipefail

apply=false

for arg in "$@"; do
  case "$arg" in
    --apply) apply=true ;;
  esac
done

dir=""
for arg in "$@"; do
  if [[ "$arg" != --* ]]; then
    dir="$arg"
    break
  fi
done

if [[ -z "$dir" ]]; then
  echo "Usage: $0 <directory> [--apply]" >&2
  exit 1
fi

if [[ ! -d "$dir" ]]; then
  echo "Error: '$dir' is not a directory." >&2
  exit 1
fi

files=$(grep -rl 'alphazero' "$dir" || true)

if [[ -z "$files" ]]; then
  echo "No files contain 'alphazero'."
  exit 0
fi

if [[ "$apply" == false ]]; then
  echo "Dry run — the following replacements would be made:"
  echo
  for f in $files; do
    grep -n 'alphazero' "$f" | while IFS= read -r line; do
      echo "$f:$line"
    done
  done
  echo
  echo "Re-run with --apply to modify files in place."
else
  for f in $files; do
    sed -i '' 's/alphazero/az/g' "$f"
    echo "Updated: $f"
    if command -v clang-format &>/dev/null; then
      clang-format -i "$f" 2>/dev/null || true
    fi
  done
  echo "Done. All occurrences replaced and clang-format applied."
fi
```
